coffeescript-idea

CoffeeScript Download Issues Source
coffeescript-idea is a plugin for IntelliJ IDEA and RubyMine that provides basic syntax highlighting for CoffeeScript. It is licensed under the Apache License 2.0.

Download

Version 0.2 was made for IntelliJ 9.0.3 Community Edition and RubyMine 2.5. It is compatible with CoffeeScript 0.9.4

Get it here.

Installation

  1. Copy the jar to your IDEA based IDE's plugins folder.
    • IntelliJ 9 Community Edition: ${HOME}/Library/Application Support/IntelliJIdea90CE
    • RubyMine 2.5: ${HOME}/Library/Application Support/RubyMine25
  2. Restart your IDEA based IDE.
  3. Check that .coffee files should now have syntax highlighting.
